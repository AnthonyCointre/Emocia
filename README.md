# Installation

To run this project locally, follow these steps:

1. Clone the Repository:

```bash
git clone https://github.com/AnthonyCointre/Emocia.git
```

2. Navigate to the project directory:
```bash
cd Emocia
```

3. Install dependencies:
```bash
pip3 install -r requirements.txt
```

4. Run the project:
```bash
python3 app.py
```

Once the application is running, go to [http://localhost:500](http://localhost:500).

# Features

- **Python** for Core Logic Powered: All appointment handling and business logic run on Python, ensuring fast, reliable, and customizable processing tailored to your workflow.
- **Flask** for Lightweight & Efficient Backend: The application is built with Flask, providing a clean, flexible, and secure backend that handles routing, form submissions, authentication, and dynamic page rendering.
- **SQLite** for Secure Data Management: SQLite ensures safe and efficient storage of appointments, messages, and user accounts, offering a stable and lightweight database solution ideal for small to medium applications.
